Site UFC Projet Web

Pour notre projet web nous avons fait un site présentant l'actualité d'un sport : l'UFC.
L'internaute aura la possibilité de visualiser la fiche des différents combattants selon leu catégorie, 
d'avoir le programme des prochains combats et il pourra également commenter les fiches et les combats.

Pour se faire nous avons utiliser le framework Django écrit en python, Gunicorn et Nginx pour le reverse proxy.

Nous avons fait notre projet sous le modèle Modèle-Vue-Template. Ainsi, dans le dossier blog vous verrez comment nous gérons nos modèles, nos vues et nos urls.
Dans le dossier templates/blog vous verrez les différents fichiers html, et dans le dossier assets se trouve nos fichiers statiques.
